# Mô tả
Dự án này xây dựng một blog listing page bằng Next.js và Tailwind CSS. Ứng dụng lấy dữ liệu bài viết từ API JSONPlaceholder, hiển thị danh sách bài viết theo dạng lưới responsive ở trang chủ, đồng thời hỗ trợ điều hướng đến trang chi tiết /blog/[id] để xem tiêu đề, nội dung đầy đủ và quay lại danh sách blog.
